# Hospital Management System

A backend REST API built using Spring Boot to manage hospital operations.

## Features
- JWT-based authentication
- DTO-based service architecture
- Global exception handling
- Patient, Doctor, Appointment management

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT
- Hibernate / JPA
- MySQL
